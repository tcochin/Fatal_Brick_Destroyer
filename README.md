# Fatal_Brick_Destroyer
A brick destroyer game made to be played on a 16*36 led pannel, powered by an arduino

Fatal Brick Destroyer est un jeu sur lequel je suis actuellement en train de travailler pour un projet arduino de mon école. La version actuellement en ligne est une version qui doit encore être retravaillé, mais le jeu en soit est déjà jouable et assez fun.
